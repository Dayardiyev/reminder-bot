# Telegram Event Scheduler Bot

Телеграм-бот, который хранит события пользователей, каждый день напоминает сколько дней осталось и в день события присылает уведомления за 10 часов, 5 часов, 30 минут, 5 минут и в момент наступления.

## Подготовка
- Установи зависимости: `python3 -m pip install -r requirements.txt`
- Создай токен у [BotFather](https://t.me/BotFather) и запиши его в переменную окружения `BOT_TOKEN`.

## Запуск
```bash
BOT_TOKEN=твой_токен python3 bot.py
```

## Команды
- Кнопки под строкой ввода: **Добавить событие**, **Список событий**.
- `/add Название; 2024-12-31 18:00` — добавляет событие в один шаг. Можно указать только дату `YYYY-MM-DD`, тогда время по умолчанию 09:00.
- `/list` — показывает список твоих событий (то же, что и кнопка).
- `/start` и `/help` — краткие подсказки.

## Как работает расписание
- Ежедневное напоминание: раз в день в `09:00` бот пишет сколько дней осталось до события.
- В день события дополнительные уведомления приходят за 10ч, 5ч, 30м, 5м и в момент начала.
- Данные сохраняются в `data/events.json`; при перезапуске бота все будущие события автоматически пересоздают свои задачи в планировщике.
